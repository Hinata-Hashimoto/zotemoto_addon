# Privacy Policy

zotemoto is a Google Sheets add-on.

This add-on does not collect, store, or share any personal data.
All data is processed only within the user's Google Spreadsheet.

The add-on makes external requests only to the public Crossref API
for the purpose of retrieving bibliographic metadata from DOI.

No data is sent to any other third-party services.

Contact:
If you have questions, please contact the developer via GitHub Issues.
